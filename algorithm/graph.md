# 图

* 连通图
    * 并查集
* 非连通图

* 有向图
    * 拓扑排序
* 无向图

### BFS 广度优先搜索
广度优先搜索是一种先访问所有同层结点，再继续深入下一层的遍历算法，用于得到无权图的最短路径。
